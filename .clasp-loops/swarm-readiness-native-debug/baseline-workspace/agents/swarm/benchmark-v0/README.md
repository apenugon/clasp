# Benchmark Wave v0

This wave is the shortest path to the first credible benchmark described in `docs/clasp-first-benchmark-slice.md`.

It intentionally does not try to advance the entire long-term roadmap in parallel.

The target is a benchmark-ready lead-inbox SaaS slice with:

- shared domain contracts
- typed routes
- generated validation
- generated route clients
- one AI-shaped boundary
- a host-rendered inbox consumer
- mirrored `Clasp` and `TypeScript` benchmark repos
